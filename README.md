# 后台管理系统

## 安装
```
npm install 或 cnpm install (推荐)
```

### 运行
```
开发环境：npm run dev
测试环境：npm run test
生产环境：npm run prod
```

### 打包
```
开发环境：npm run build:dev
测试环境：npm run build:test
生产环境：npm run build
```

### 更改接口地址
```
开发环境：文件夹： .env.development，变量名：VUE_APP_API
测试环境：文件夹： .env.test，变量名：VUE_APP_API
生产环境：文件夹： .env.production，变量名：VUE_APP_API
```
